# C-DNI-PRACTISE
Un total de 10 practicas dónde aprendemos las bases del lenguaje C, en uso práctico de creación de DNIs Españoles

"Una colección de 10 ejercicios progresivos en C para entender conceptos fundamentales (strings, bucles, funciones) mediante la generación y validación de DNIs/NIEs españoles. Desde el uso básico de printf/scanf hasta la implementación con ficheros y structs"
